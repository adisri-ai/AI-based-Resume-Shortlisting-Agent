# AI-based-Resume-Shortlisting-Agent
An Agentic AI project that can be used for shortlisting candidates based on a provided JD and their resume. It scores the resume/CVs of each candidate based on relevance to the provided JD.
Currently deployed as an Azure Blob Trigger Function which gets triggered on user uploading a file into the required blob directory.
Uses Opneai API for grading/classification.
